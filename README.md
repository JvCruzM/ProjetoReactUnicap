# ProjetoReactUnicap
Este projeto foi desenvolvido como introdução ao **React**, utilizando **Vite** para criação e execução do ambiente de desenvolvimento.  
O objetivo é demonstrar conceitos fundamentais como **componentes**, **JSX**, **hooks** e especialmente o uso do **`useState`** para gerenciar estados dentro de um componente React.
